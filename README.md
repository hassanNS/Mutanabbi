# Mutanabbi - Online Arabic Editor with AI Suggestions

## Overview
Mutanabbi is an online Arabic text editor that provides real-time AI-powered suggestions to enhance the writing experience. Built with Next.js, this application aims to assist Arabic Students to improve their prose and writing skill.

## Features
- AI-powered suggestions for vocabulary, grammar, and style
- Modern, responsive user interface

## Tech Stack
- Next.js
- React
- TypeScript
- [AI model integration]

## Getting Started

### Prerequisites
- Node.js (v14.0.0 or later)
- npm or yarn

### Installation
```bash
# Clone the repository
git clone https://github.com/yourusername/mutanabbi-nextjs.git

# Navigate to project directory
cd mutanabbi-nextjs

# Install dependencies
npm install
# or
yarn install
```

### Development
```bash
# Run the development server
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Building for Production
```bash
# Build the application
npm run build
# or
yarn build

# Start the production server
npm start
# or
yarn start
```
