# مُحسِّن النص العربي

أداة قوية لتحليل وتحسين النصوص العربية باستخدام الذكاء الاصطناعي. تساعدك على كتابة نصوص أكثر وضوحًا وقوة من خلال تحليل النحو والأسلوب والترجمة.

## المميزات

- **تحليل فوري للنص**: احصل على إحصائيات مفصلة عن نصك
- **تصحيح نحوي ذكي**: اكتشاف الأخطاء النحوية والإملائية باستخدام الذكاء الاصطناعي
- **تحليل الأسلوب**: تحديد الجمل الطويلة والعبارات الضعيفة والاستخدام المفرط للظروف
- **ترجمة تلقائية**: ترجمة فورية للنص إلى الإنجليزية
- **واجهة عصرية**: تصميم متجاوب مع دعم الوضع المظلم والفاتح

## التقنيات المستخدمة

- **Next.js 14** - إطار عمل React مع App Router
- **TypeScript** - لكتابة كود آمن ومنظم
- **Tailwind CSS** - للتصميم السريع والمتجاوب
- **Gemini AI API** - للتحليل النحوي والترجمة

## التثبيت والتشغيل

1. **تثبيت المكتبات**:
   \`\`\`bash
   npm install
   \`\`\`

2. **إعداد متغيرات البيئة**:
   أنشئ ملف \`.env.local\` وأضف مفتاح Gemini API:
   \`\`\`
   NEXT_PUBLIC_GEMINI_API_KEY=your_api_key_here
   \`\`\`

3. **تشغيل الخادم المحلي**:
   \`\`\`bash
   npm run dev
   \`\`\`

4. **افتح المتصفح** وانتقل إلى: http://localhost:3000
